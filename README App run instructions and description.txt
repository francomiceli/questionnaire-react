// DESCRIPTION //

This is a simple questionnaire app, where you can answer a series of questions and see them summarized at the end. 
It is step-driven, containing a series of views where you can go back and forth between the questions.
It is responsive and supports Dark Mode with a toggle on the upper side.

// INSTRUCTIONS //

To run the React application in your browser, please follow these steps:

1. Extract the 'questionnaire-react' folder from the 'questionnaire-react.rar' file.

2. Open a terminal tab and navigate to the 'questionnaire-react' directory (the place you have extracted it).

3. Write and execute these commands in the following order, one at a time (without the quotation marks):

	A. 'npm install'
	
	B. 'npm start'
